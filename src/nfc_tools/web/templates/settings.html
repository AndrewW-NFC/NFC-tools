Recording format

Auto now prefers the sounddevice/CoreAudio backend on macOS because testing showed ffmpeg/avfoundation introduced recurring short spikes while sounddevice matched the clean DAW path.

Auto/native preserves the device's incoming format when possible. On macOS this commonly means 48 kHz / 32-bit float, which is the closest match to DAW-style recording.

For uncompressed WAV, bitrate is a result of sample rate, bit depth, and channels. Choose a format preset rather than a target bitrate.
